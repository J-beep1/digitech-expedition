Syntax Sheet

Python + pygame — keep this open while you code

Core Python
Works in any Python project — text RPG, dashboard, or pygame game
Variables store a value
Give a value a name so you can use it later. No declaration needed — just assign.
player_health = 100
name = "Zara"
is_alive = True
speed = 3.5
if / elif / else state-check
One branch fires — never both. elif = "else if". Indentation is the structure.
if hp <= 0:
    print("You died")
elif hp < 20:
    print("Low health!")
else:
    print("Still fighting")
for loop fixed repeats
Runs once per item in a list, or a set number of times with range().
for enemy in enemies:
    print(enemy)

for i in range(5): # 0,1,2,3,4
    print(i)
while loop unknown repeats
Keeps looping until the condition is false. Use when you don't know how many turns the game will take.
while hp > 0 and dist > 0:
    hp -= damage
    dist -= speed
# exits when either condition fails
Functions reuse code
Give a block of code a name so you can call it anywhere. Use return to send a value back out.
def attack(damage, target_hp):
    target_hp -= damage
    return target_hp

enemy_hp = attack(15, enemy_hp)
Lists ordered collection
Ordered, changeable. Index starts at 0. Use .append() to add, len() to count.
inventory = ["sword", "potion"]
inventory.append("shield")
print(inventory[0]) # "sword"
print(len(inventory)) # 3
Dictionaries named pairs
Key → value pairs. Look up by name, not position. Great for character stats, room data, item properties.
player = {"hp": 80, "xp": 120}
print(player["hp"]) # 80
player["hp"] -= 10 # take damage
player["level"] = 2 # add new key
User input text RPG
input() always returns a string. Wrap in int() if you need a number.
choice = input("Attack or flee? ")
if choice == "attack":
    print("You swing your sword!")

age = int(input("Enter age: "))

pygame
Adds a window, graphics, movement, and input — needed for maze, snake, shooter
⚠
Import pygame first — every pygame project starts with the same three lines. If you forget pygame.init() or the game loop, nothing works. Copy the skeleton below and build from it.
The pygame skeleton — every project starts here copy this
import pygame
pygame.init()

# window setup
WIDTH, HEIGHT = 800, 600
screen = pygame.display.set_mode((WIDTH, HEIGHT))
pygame.display.set_caption("My Game")
clock = pygame.time.Clock()
FPS = 60

# colours (R, G, B)
BLACK = (0, 0, 0)
WHITE = (255, 255, 255)
PINK = (255, 46, 154)
# game loop
running = True
while running:

    # 1 — handle events
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = False

    # 2 — update game state
    # (move player, check collisions, etc)

    # 3 — draw everything
    screen.fill(BLACK)
    # draw shapes here
    pygame.display.flip()
    clock.tick(FPS)

pygame.quit()
Drawing shapes pygame.draw
All draw calls go between screen.fill() and display.flip(). Rect = (x, y, width, height).
# rectangle
pygame.draw.rect(screen, WHITE, (100,200,50,50))
# circle: (surface, colour, centre, radius)
pygame.draw.circle(screen, PINK, (400,300), 20)
# line: start, end, thickness
pygame.draw.line(screen, WHITE, (0,0), (100,100), 2)
Keyboard input movement
get_pressed() checks keys held down — use in the update section of the game loop.
keys = pygame.key.get_pressed()
if keys[pygame.K_LEFT]:
    player_x -= speed
if keys[pygame.K_RIGHT]:
    player_x += speed
# also: K_UP K_DOWN K_SPACE K_a K_w
Collision detection hit check
pygame.Rect objects can check if they overlap with .colliderect(). Define rects for player and enemies.
player_rect = pygame.Rect(px,py,40,40)
enemy_rect = pygame.Rect(ex,ey,40,40)
if player_rect.colliderect(enemy_rect):
    hp -= 10
Displaying text HUD / UI
Create a font object once, then use .render() each frame. Text must be a string — wrap numbers in str().
font = pygame.font.SysFont(None, 36)
text = font.render("HP: " + str(hp), True, WHITE)
screen.blit(text, (10, 10))
# blit = "draw this onto the screen"
Loading a sprite image Art cross-over
Load your sprite image file (PNG with transparency works best). Scale it to fit your game's tile or character size.
img = pygame.image.load("player.png")
img = pygame.transform.scale(img, (40,40))
# then in the game loop:
screen.blit(img, (player_x, player_y))
Boundary checking keep in bounds
Stop player or objects from moving off screen. Clamp the position after movement — pattern used in every movement game.
player_x += speed
# clamp: don't go past edges
if player_x < 0:
    player_x = 0
if player_x > WIDTH - tile:
    player_x = WIDTH - tile
