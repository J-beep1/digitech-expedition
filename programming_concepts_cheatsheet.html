int — whole number float — decimal str — text bool — True/False list — ordered collection dict — key→value pairs

Loops

for — iterate over a sequence

for item in list:
  do_something(item)

while — repeat while a condition is True

while count < 10:
  count += 1

Functions

Reusable blocks of code. Define once, call many times.

→

→
return value
def greet(name):
  return "Hi, " + name
  • Parameters — inputs the function receives
  • Return value — what the function sends back
  • Scope — variables inside a function are local

Data structures

List [ ]

Ordered, indexed from 0. Add with .append(), remove with .remove().

Dictionary { }

Key→value pairs. Access with d["key"]. Fast lookup.

String " "

Sequence of characters. .upper() .split() len()

Debugging

Common errors

  • SyntaxError — invalid code structure
  • NameError — variable not defined
  • TypeError — wrong data type used
  • IndexError — list index out of range

read error msg
→

Big ideas to remember

  • Decomposition — break big problems into smaller ones
  • Abstraction — hide detail, show only what matters
  • Pattern recognition — spot similarities to reuse solutions
  • DRY — Don't Repeat Yourself → use functions
  • Comments — explain why, not just what
  • Test edge cases — what if input is 0, empty, or huge?
